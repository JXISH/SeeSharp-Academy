# SeeSharp-Academy
SeeSharp Open Academy source code





## 目录结构

* Courses： 课程资料
  * 0.0.3 宣讲会：宣讲会的demo
  * 0.1.0 DAQ101: 第一课基础采集-分析-展示存盘的起步代码和完成后代码
  * Help：相关帮助文档，比如名词解释
* Gallery: 演示和范例
  * Data
    * Blade Ping：2种采集硬件对两片喷气发动机扇叶敲击的声音采集数据，可以用SpectrumViewer观看分析
  * ChromeTodo List：AI生成的Chrome浏览器插件，功能是待办事项管理器
  * Markdown Viewer：AI生成的网页版markdown文件浏览器
  * Plot Data： AI生成的一位数组可视化浏览器，可用于Console软件输出逗号分隔数据的可视化
  * SpectrumViewer：面向EasyChartX右键输出的csv数据，读取和频谱分析。可以浏览Blade Ping数据
