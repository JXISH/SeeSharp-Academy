## 1. 项目概述

Spectrum Viewer 是一个专业的**频谱分析工具套件**，用于对时域信号进行全面的频域分析和频率检测。该工具集成了多个子模块，提供了从数据加载到高级频率分析的完整解决方案。

### 主要特点
- ✅ 支持从CSV文件加载波形数据
- ✅ 时域波形显示与光标切割功能
- ✅ 多种频谱分析方法（功率谱、倒谱）
- ✅ 联合时频分析（JTFA）可视化
- ✅ 基于Matrix Pencil算法的高精度频率检测
- ✅ 灵活的信号预处理（降采样、滤波）

### 应用领域
- 振动分析与故障诊断
- 音频信号处理
- 通信信号分析
- 电力系统谐波分析
- 科研数据可视化

## 2. 系统架构

Spectrum Viewer 由三个相互关联的Windows Forms应用程序组成：

```
Spectrum Viewer
├── Spectrum Viewer.exe          # 主程序：频谱分析器
├── CSV Loader.exe               # 数据加载工具
└── Peak Frequency Finder.exe    # 频率检测工具
```

### 模块关系
```
┌──────────────────────┐
│   CSV Loader         │  ← 用户加载数据
│  (数据加载模块)       │
└──────────┬───────────┘
           │ CSV数据
           ↓
┌──────────────────────┐
│  Spectrum Viewer     │  ← 主分析界面
│  (频谱分析主模块)     │
└──────────┬───────────┘
           │ 切割的波形段
           ↓
┌──────────────────────┐
│ Peak Frequency Finder│  ← 高精度频率检测
│ (频率检测模块)        │
└──────────────────────┘
```

## 3. 功能模块

### 3.1 Spectrum Viewer（主程序）
**核心功能：**
- 时域波形显示
- 光标切割波形段
- 频谱分析（功率谱/倒谱）
- 联合时频分析（JTFA）
- 调用频率检测工具

**支持的图表类型：**
- 时域波形图
- 频域幅度谱
- 时频热力图（2D Spectrogram）

### 3.2 CSV Loader
**核心功能：**
- 从CSV文件加载波形数据
- 自动识别采样率
- 数据预览和验证

**支持的CSV格式：**
```
时间,幅值
0.0000,1.2345
0.0001,2.3456
...
```
或单列幅值数据

### 3.3 Peak Frequency Finder
**核心功能：**
- 信号预处理（降采样、滤波）
- FFT频谱计算
- Matrix Pencil频率估计
- 频率分量智能筛选

**算法优势：**
- 频率精度高于传统FFT峰值检测
- 适用于多分量信号
- 对短数据和低信噪比信号效果好

## 5. Spectrum Viewer 使用指南

### 5.1 界面布局

![](.\Spectrum Viewer_Blade02_Screenshot.PNG)

### 5.2 基本操作流程

#### 步骤1：加载波形数据
1. 点击 **[Load]** 按钮
2. CSV Loader对话框打开
3. 选择CSV文件并配置参数（详见6. CSV Loader使用指南）
4. 点击"加载"完成数据导入

**提示：** 加载成功后，窗口标题会显示当前文件名

#### 步骤2：调整采样率
- 在 **Sample Rate** 输入框中设置正确的采样率（Hz）
- 采样率应与CSV文件中数据的实际采样率一致

#### 步骤3：切割波形段
1. 在时域波形图中移动**光标1**和**光标2**
2. 两个光标之间的波形段会自动显示在"切割波形段"区域
3. 调整 **Window Length** 控制切割窗口大小

**应用场景：**
- 分析信号的特定时间段
- 选取平稳段进行频谱分析
- 观察瞬态信号的频谱变化

#### 步骤4：频谱分析

##### 4.1 功率谱分析
1. 确保 **☑ Cepstrum** 未勾选
2. 勾选 **☑ Spectrum dB** 显示对数幅度（dBV）
3. 频谱图自动更新

**功率谱显示内容：**
- X轴：频率（Hz）
- Y轴：幅度（V² 或 dBV）

##### 4.2 倒谱分析（Cepstrum）
1. 勾选 **☑ Cepstrum**
2. 系统自动计算并显示倒谱

**倒谱应用：**
- 检测信号的谐波周期性
- 回声检测和延迟估计
- 音高检测

#### 步骤5：联合时频分析（JTFA）

##### 5.1 参数设置
- **Start Time (s)：** 分析起始时间
- **Duration (s)：** 分析持续时间
- **Window Length：** FFT窗口大小（影响频率分辨率）
- **Window Types：** 窗函数类型（汉宁、汉明等）
- **Color Type：** 热力图颜色方案

##### 5.2 执行分析
1. 点击 **[JTFA]** 按钮
2. 等待计算完成
3. 查看时频热力图

**热力图解读：**
- X轴：频率
- Y轴：时间（从上到下）
- 颜色：功率谱密度（dB）
- 亮色表示高能量，暗色表示低能量

**颜色方案推荐：**
- `Turbo` / `Jet`：通用分析，高对比度
- `Viridis` / `Plasma`：色盲友好，感知均匀
- `Grayscale`：黑白打印

#### 步骤6：频率检测
1. 在切割波形段后，点击 **[Find Frequencies]** 按钮
2. Peak Frequency Finder窗口打开（详见7. Peak Frequency Finder使用指南）
3. 在频率查找器中完成高精度频率检测

## 6. CSV Loader 使用指南

### 6.1 界面说明

![](.\CSV Loader_main.PNG)

### 6.2 支持的CSV格式

#### 格式1：时间-幅值两列
```csv
Time,Amplitude
0.0000,1.2345
0.0001,2.3456
0.0002,1.5432
...
```

#### 格式2：单列幅值
```csv
Amplitude
1.2345
2.3456
1.5432
...
```

#### 格式3：多列数据
```csv
Time,Ch1,Ch2,Ch3
0.0000,1.2,2.3,3.4
0.0001,1.3,2.4,3.5
...
```

### 6.3 操作步骤

#### 步骤1：选择文件
1. 点击 **[Browse...]** 按钮
2. 选择CSV文件
3. 数据预览自动显示

#### 步骤2：配置参数

##### 时间列设置
- **使用时间列：** 勾选此项表示CSV第一列是时间数据
  - 系统会根据时间差自动计算采样率
  - 适用于非均匀采样数据
  
- **仅幅值列：** 勾选此项表示CSV仅包含幅值数据
  - 需要手动设置采样率
  - 适用于均匀采样数据

##### 采样率设置
- 如果"使用时间列"：采样率自动计算（可手动调整）
- 如果"仅幅值列"：必须手动输入采样率

##### 跳过行数
- 设置跳过文件开头的行数（如标题行）
- 默认为0

#### 步骤3：加载数据
1. 确认配置无误后，点击 **[加载]**
2. 数据自动传递给Spectrum Viewer
3. CSV Loader对话框关闭

### 6.4 数据验证

加载前请检查：
- ✅ 数据列数正确
- ✅ 采样率合理
- ✅ 数据范围正常
- ✅ 没有缺失值或异常值

## 7. Peak Frequency Finder 使用指南

### 7.1 界面说明

![](.\Frequency Finder_Blade02_Screenshot.PNG)

### 7.2 工作流程

#### 步骤1：获取数据
- **方式1：** 从Spectrum Viewer传递（自动加载切割的波形段）
- **方式2：** 点击 **[Load]** 手动加载CSV文件

#### 步骤2：设置频率范围
- **Start Freq (Hz)：** 起始频率
- **Stop Freq (Hz)：** 停止频率

**频率范围设置原则：**
- 最高频率不能超过采样率的50%（奈奎斯特频率）
- 起始频率应高于直流分量
- 根据信号特征选择合理的分析带宽

#### 步骤3：配置分析参数

##### Relative Threshold（相对阈值）
- **范围：** 0.0 ~ 1.0（建议值：0.1）
- **含义：** 相对于最大幅度的阈值
- **作用：** 控制Matrix Pencil算法的信号阶数
  - 较小值：检测更多频率分量（可能包含噪声）
  - 较大值：仅检测主要频率分量

##### Max Peak Num（最大峰值数）
- **范围：** 1 ~ 50（建议值：13）
- **含义：** 最多显示的频率分量数量
- **作用：** 限制结果数量，避免显示过多微小分量

##### Y Axis Log（Y轴对数坐标）
- 勾选：使用对数Y轴显示频谱
- 适用于观察宽动态范围的信号

#### 步骤4：执行分析
1. 点击 **[Analysis]** 按钮
2. 查看进度条和提示信息
3. 等待分析完成

**分析过程：**
```
0-5%：   预处理降采样
5-15%：  设计滤波器
15-20%： 计算FFT频谱
20-80%： Matrix Pencil频率检测
80-100%： 结果整理
```

#### 步骤5：查看结果

##### 5.1 频谱图
- **蓝色曲线：** FFT频谱
- **红色散点：** Matrix Pencil检测到的频率点
- 勾选/取消 **☑ Y Axis Log** 切换Y轴显示模式

##### 5.2 结果表格
- **#：** 序号
- **Frequency (Hz)：** 频率值
- **Amplitude：** 幅度（与FFT频谱对应）

##### 5.3 导出数据
- 控制台会自动输出CSV格式结果
- 可复制到Excel等工具进行进一步分析

### 7.3 算法说明

#### Matrix Pencil算法
一种基于子空间分解的高精度频率估计方法。

**算法步骤：**
1. 构建Hankel矩阵
2. 奇异值分解（SVD）
3. 确定信号阶数
4. 特征值分解
5. 计算频率

**优势：**
- 精度高于FFT峰值检测
- 不受频率栅栏限制
- 适用于短数据
- 对低信噪比信号鲁棒

**适用场景：**
- 多频率分量信号
- 频率接近的信号
- 指数衰减信号
- 调制信号分析

#### 预处理流程
1. **降采样：** 降低数据量，提高计算效率
2. **滤波：** 去除目标频段外的噪声和干扰
   - 根据频率范围自动选择滤波器类型
   - 支持带通、高通、低通滤波

### 7.4 参数调优建议

#### 场景1：检测主要频率
```
Relative Threshold: 0.1  (10%)
Max Peak Num: 13
```

#### 场景2：检测所有频率分量
```
Relative Threshold: 0.01  (1%)
Max Peak Num: 50
```

#### 场景3：低信噪比信号
```
Relative Threshold: 0.2  (20%)
Max Peak Num: 10
预处理滤波：使用更窄的带宽
```

#### 场景4：密集频率分量
```
Relative Threshold: 0.05  (5%)
Max Peak Num: 30
确保足够高的频率分辨率
```

## 8. 典型应用场景

### 8.1 旋转机械故障诊断
**目的：** 检测轴承、齿轮等旋转部件的故障频率

**操作步骤：**
1. 加载振动传感器数据
2. 设置频率范围为0-1000 Hz
3. 执行频率检测
4. 识别特征频率（转频、倍频、边频带）

**预期结果：**
- 清晰的转频及其谐波
- 故障特征频率（如轴承缺陷频率）

### 8.2 音频信号分析
**目的：** 分析音频信号的基频和谐波

**操作步骤：**
1. 加载音频波形
2. 设置频率范围为20 Hz - 20 kHz
3. 使用JTFA观察频率随时间的变化
4. 使用倒谱检测音高和回声

**预期结果：**
- 基频识别
- 谐波结构
- 音高变化轨迹

### 8.3 电力系统谐波分析
**目的：** 检测电力信号的谐波分量

**操作步骤：**
1. 加载电压/电流波形
2. 设置频率范围为0-2 kHz
3. 执行频率检测
4. 分析各次谐波含量

**预期结果：**
- 基波（50/60 Hz）
- 各次谐波（2次、3次、5次等）
- 谐波总畸变率（THD）

### 8.4 通信信号分析
**目的：** 分析调制信号的频率成分

**操作步骤：**
1. 加载已调制信号
2. 使用JTFA观察时频特性
3. 检测载波频率和调制边带
4. 分析调制深度

**预期结果：**
- 载波频率
- 调制边带
- 时频分布特征

## 9. 常见问题

### Q1: 为什么加载CSV后显示为直线？
**A:** 可能原因：
- 采样率设置错误（相差数量级）
- 数据列选择错误
- 数据格式不正确

**解决方法：**
- 检查采样率是否正确
- 在CSV Loader中预览数据
- 确认选择了正确的数据列

### Q2: 频谱图看不到明显的峰值？
**A:** 可能原因：
- 信号本身没有明显的周期性
- 噪声过大淹没信号
- 频率范围设置不当

**解决方法：**
- 尝试勾选"Spectrum dB"查看对数谱
- 调整频率范围
- 使用预处理滤波降低噪声

### Q3: Matrix Pencil检测不到频率？
**A:** 可能原因：
- Relative Threshold 设置过高
- 信号质量差
- 频率超出搜索范围

**解决方法：**
- 降低 Relative Threshold（如从0.1改为0.01）
- 检查频率范围设置
- 改善信号质量（滤波）

### Q4: JTFA热力图显示异常？
**A:** 可能原因：
- 分析时间段数据不足
- 窗口长度设置不当
- 内存不足

**解决方法：**
- 增加Duration（分析持续时间）
- 调整Window Length（建议256-1024）
- 关闭其他程序释放内存

### Q5: 如何提高频率检测精度？
**A:** 优化建议：
1. 使用更长的数据段
2. 降低采样率（相对提高频率分辨率）
3. 调整Relative Threshold
4. 预处理滤波去除噪声

### Q6: 导出的数据格式是什么？
**A:** 控制台输出格式为CSV：
```
Frequency_Hz,Amplitude_FFT
50.0000,1.23456789
100.0000,5.67890123
```

可复制到Excel、MATLAB、Python等工具进行进一步分析。

### Q7: 支持哪些采样率？
**A:** 理论上支持任意采样率，但：
- 最低采样率：应大于最高分析频率的2倍
- 推荐范围：1 Hz - 1 MHz
- 常用采样率：
  - 音频：44.1 kHz, 48 kHz
  - 振动：10 kHz - 100 kHz
  - 电力：10 kHz - 50 kHz

### Q8: 如何批量处理多个文件？
**A:** 当前版本需要逐个文件手动处理，自动化批量处理功能计划在后续版本中添加。

### Q9: 可以保存分析结果吗？
**A:** 当前版本结果通过控制台输出，可以：
- 复制控制台文本保存
- 截图保存图表
- 后续版本将添加导出功能

### Q10: Matrix Pencil与FFT有什么区别？
**A:** 对比：

| 特性 | FFT峰值检测 | Matrix Pencil |
|------|-------------|---------------|
| 频率精度 | 受频率栅栏限制 | 理论精度更高 |
| 幅度信息 | 直接提供 | 通过FFT插值获取 |
| 计算速度 | 快 | 较慢 |
| 数据长度要求 | 较长 | 可用于短数据 |
| 频率分辨率 | Δf = fs/N | 超分辨率 |
| 适用场景 | 快速分析 | 高精度分析 |

**建议：**
- 快速浏览：使用FFT
- 精确检测：使用Matrix Pencil

## 附录：快捷键与技巧

### 频率查找技巧
1. **数据预处理**：Matrix Pencil比较占资源，建议分析数据长度不要超过2000点。建议在Spectrum Viewer预先观察好信号频谱特点，选择合适的信号片段。频谱分析的片段就是频率查找的波形片段。如果你感兴的频率相对能奎斯特频率低很多，请在频率查找界面先设置合适的起始和终止频率。你点击分析后，软件先做数字降采样和带通滤波，再做Matrix Pencil分析。假设你的信号采样率25.6k，只对200-800Hz频率感兴趣，软件会根据上限频率2.5倍为理想采样率，对信号做整数降采样，也就是降采样12倍。对于原始采样12800长度的片段，在这个设置下，实际Matrix Pencil数据长度1066点。
2. **频率范围**：根据实际情况合理配置，尽量不要包括DC，如果StopFrequency设置低于StartFrequency，软件忽略StopFrequency，只做高通滤波。
3. **参数调整**：
   * Max Peak Number：最多输出的峰值频率数量，如果你要的频点没有检出，可以增加一些试试。
   * Relative Threshold：建议从0.1开始从大到小尝试。这个大约是检出最低幅度较最大幅度的相对比例。
4. **结果验证**：对比FFT和Matrix Pencil结果
5. **数据保存**：及时复制软件输出
