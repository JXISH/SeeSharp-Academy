### Spectrum Viewer uses JXI DSP-Core

you need to:

1. install a certain Intel MKL， get it from JYPedia. JYPedia is an excel file you can download from jytek.com

   JYPedia>>Drivers and Software >> JXDSPRuntimeMKL2017.1.143_x64

2. jxiIPP.dll, you can download from github >> DSP-Core \ Dependency  by JXISH

   github.com/JXISH/Seesharp-DSP-Core-Lib/tree/main/Development/CSharp/Core%20Library/Public/External/Dependency/x64

   * you can copy this dll to your windows\system32.  It is used by all DSP-Core based applications
