# SyncFolders 使用说明

## 简介

SyncFolders 是一个文件夹同步工具，用于比较两个目录，找出 **目录 A 中存在但目录 B 中不存在** 的文件，并支持用户选择性地将这些文件复制到目录 B。

适用于文件迁移、备份校验、多版本目录合并等场景。

---

## 运行方式

```bash
python sync_folders.py
```

需要 Python 3.8+，无需安装额外依赖（使用内置 tkinter）。

## 操作步骤

### 1. 选择目录

- **目录 A（源）**：包含你希望同步的文件的目录
- **目录 B（目标）**：你希望将缺失文件复制到的目录

点击 **"浏览…"** 按钮选择，或直接在输入框中粘贴路径。

### 2. 点击"比较"

程序会后台扫描两个目录下的所有文件（含子目录），完成后在列表中显示 **A 中独有** 的文件。

- 文件按 **子目录分组**，点击 `📁` 可展开/折叠
- 状态栏显示独有文件总数

### 3. 选择文件

- **单选**：点击某个文件
- **多选**：按住 `Ctrl` 逐个点击，或按 `Shift` 连续选择
- **选中整个文件夹**：点击文件夹节点，其下所有文件都会被选中
- **全选**：点击"全选"按钮选中所有独有文件
- **取消全选**：点击"取消全选"按钮

### 4. 复制文件

点击 **"📋 复制选中项到 B"**，确认弹窗后开始复制。

- 自动在 B 中创建对应的子目录结构
- 保留文件的修改时间等元数据（使用 `shutil.copy2`）
- 复制成功的文件状态变为 **"✅ 已复制"**
- 失败的文件会在日志中记录原因

---

## 右键菜单

在文件列表中右键点击，可弹出快捷菜单执行复制操作。

---

## 注意事项

- 比较基于 **文件相对路径**（含文件名），不比较文件内容
- 如果 A 和 B 中文件名相同但内容不同，不会出现在差异列表中
- 大量文件时扫描可能需要几秒，进度条会实时反馈
- 复制操作不会删除或覆盖 B 中已有的文件
- 路径支持中文和空格

## 典型使用场景

| 场景                 | 说明                                                 |
| -------------------- | ---------------------------------------------------- |
| 项目备份校验         | 比较工作目录和备份目录，找出未备份的文件             |
| 多版本目录合并       | 找出旧版本中有但新版本中遗漏的文件                 |
| U盘/移动硬盘同步     | 比较本地目录和移动存储，选择性复制新增文件           |
| 服务器部署差异排查   | 比较本地源码和部署目录，找出缺失文件                 |
