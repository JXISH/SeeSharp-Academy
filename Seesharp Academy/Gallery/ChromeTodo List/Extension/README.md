# Chrome 待办事项插件

一个简单而强大的Chrome浏览器待办事项管理插件。

## 功能特性

- ✅ 添加新任务，设置优先级(0-5)、预计用时和完成期限
- ✅ 任务完成状态切换
- ✅ 右键菜单编辑和删除任务
- ✅ 按状态筛选（全部/未完成/已完成）
- ✅ 多种排序方式（按优先级/按时间/按期限）
- ✅ 紧凑的列表显示
- ✅ 可折叠的添加任务区域
- ✅ 数据本地存储，不会丢失
- ✅ 统计信息显示
- ✅ 期限提醒（逾期任务红色闪烁，24小时内橙色显示）

## 安装方法

### 步骤1: 准备图标文件

由于图标文件需要是PNG格式，您可以使用以下方法之一：

**方法1: 使用在线工具生成图标**
- 访问 https://www.favicon-generator.org/ 或类似网站
- 上传任意图片或使用默认图标
- 生成16x16, 48x48, 128x128三个尺寸的PNG图标
- 保存为 `icon16.png`, `icon48.png`, `icon128.png` 到插件目录

**方法2: 使用简单的纯色图标（快速方案）**
- 创建三个纯色的PNG图标文件
- 或使用以下Python脚本快速生成：

```python
from PIL import Image, ImageDraw, ImageFont

# 创建简单的纯色图标
def create_icon(size, filename):
    img = Image.new('RGBA', (size, size), (102, 126, 234, 255))
    draw = ImageDraw.Draw(img)
    draw.rectangle([size//4, size//4, size*3//4, size*3//4], fill='white')
    img.save(filename)

create_icon(16, 'icon16.png')
create_icon(48, 'icon48.png')
create_icon(128, 'icon128.png')
```

### 步骤2: 加载插件到Chrome

1. 打开Chrome浏览器
2. 在地址栏输入 `chrome://extensions/`
3. 打开右上角的"开发者模式"开关
4. 点击"加载已解压的扩展程序"
5. 选择这个文件夹 (ChromeTodo 01)
6. 插件安装完成！

## 使用说明

### 添加任务
1. 点击浏览器工具栏的插件图标
2. 点击"+ 添加新任务"按钮展开输入区域
3. 在输入框中输入任务描述
4. 设置优先级 (0-5，其中5为最高优先级)
5. 设置预计用时（小时）
6. 设置完成期限（可选，精确到分钟）
7. 点击"添加任务"或按回车键（添加后自动收起）

### 管理任务
- **完成任务**: 勾选任务前的复选框
- **编辑任务**: 右键点击任务，选择"✏️ 编辑"（可修改所有属性包括期限）
- **删除任务**: 右键点击任务，选择"🗑️ 删除"
- **筛选任务**: 使用顶部按钮筛选全部/未完成/已完成的任务
- **排序任务**: 使用右侧下拉菜单选择按优先级/时间/期限排序

### 期限提醒
- **正常显示**: 灰色文字，距离期限还有较长时间
- **紧急提醒**: 橙色加粗，24小时内到期
- **逾期提醒**: 红色加粗闪烁，已超过期限

### 优先级说明
- 0: 灰色 - 无优先级
- 1: 蓝色 - 低优先级
- 2: 绿色 - 较低优先级
- 3: 黄色 - 中等优先级
- 4: 橙色 - 较高优先级
- 5: 红色 - 最高优先级

### 统计信息
底部显示：
- 任务总数
- 已完成/待完成任务数
- 总计用时和已完成用时

## 技术栈

- HTML5
- CSS3 (渐变背景，响应式设计)
- Vanilla JavaScript
- Chrome Extension API (storage)
- LocalStorage for data persistence

## 注意事项

- 任务数据存储在浏览器本地存储中
- 清除浏览器数据可能会导致任务丢失
- 建议定期备份重要任务
