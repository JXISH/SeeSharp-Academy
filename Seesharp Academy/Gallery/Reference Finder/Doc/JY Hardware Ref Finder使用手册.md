# 硬件AI训导词生成器 使用手册

## 简介

硬件AI训导词生成器（HW AI Guide Generator）是一个跨平台桌面应用程序，帮助用户快速生成针对简仪硬件的C#编程AI训导词。通过输入硬件型号，自动查找对应的驱动文件和编程范例，生成可直接用于AI编程助手的训导词。

## 系统要求

- **操作系统**: Windows 10/11 或 Linux
- **运行时**: .NET 8.0 Runtime
- **依赖**: 需要已安装简仪硬件驱动和范例

## 界面说明

### 输入设置区域

| 控件 | 说明 |
|------|------|
| JYPEDIA文件 | 选择JYPEDIA Excel文件路径（.xlsx），包含驱动和范例信息 |
| 操作系统 | 选择目标操作系统（Windows/Linux），默认为当前系统 |
| 硬件型号 | 输入硬件型号，如"JYUSB1601"或"1601" |
| 查找按钮 | 根据输入查找驱动和范例 |

### 复制驱动选项

- 勾选"复制驱动到目标目录"后，可指定训导词包含驱动文件的目标存放路径
- 生成的训导词会包含驱动文件复制指令

### 范例选择

- 查找完成后，左侧树形控件显示找到的范例项目
- 勾选需要参考的范例（可多选）
- 选中的范例会被包含在生成的训导词中

### 输出区域

- 显示搜索结果摘要和生成的AI训导词
- 点击"生成AI训导词"生成完整训导词
- 点击"复制到剪贴板"复制内容

### 历史记录

- 自动保存最近20条输入和输出记录
- 点击历史记录项可快速查看之前的训导词

## 使用步骤

### 1. 选择JYPEDIA文件

点击"浏览..."按钮，选择JYPEDIA Excel文件。该文件通常位于简仪安装目录中。

### 2. 输入硬件型号

在"硬件型号"输入框中输入要编程的硬件型号。支持以下格式：
- 完整型号：`JYUSB1601`
- 简写数字：`1601`

程序会自动提取4-5位数字作为型号数值进行匹配。

### 3. 点击查找

点击"查找"按钮，程序将：
1. 在驱动根目录搜索包含型号数字的驱动DLL和XML文件
2. 在范例根目录搜索包含型号数字和"Example"的目录

### 4. 选择范例（可选）

在左侧范例树中勾选想要参考的范例项目。

### 5. 生成训导词

点击"生成AI训导词"按钮，生成包含以下内容的训导词：
- 硬件型号声明
- 驱动引用指令
- XML文档参考路径
- 范例代码参考路径

### 6. 复制使用

点击"复制到剪贴板"，将训导词粘贴到AI编程助手中使用。

## 目录说明

### 驱动根目录

| 操作系统 | 路径 |
|----------|------|
| Windows | `C:\SeeSharp\JYTEK\Hardware` |
| Linux | `~/SeeSharp/JYTEK/Hardware` |

### 范例根目录

| 操作系统 | 用户目录 | 系统目录 |
|----------|----------|----------|
| Windows | `C:\Users\用户名\SeeSharp\JYTEK` | `C:\SeeSharp\JYTEK` |
| Linux | `/home/用户名/SeeSharp/JYTEK` | `/home/SeeSharp/JYTEK` |

程序优先搜索用户目录，再搜索系统目录。

## 历史记录存储位置

| 操作系统 | 路径 |
|----------|------|
| Windows | `C:\Users\用户名\AppData\Roaming\SeeSharp\JYTEK\AIGuideGenerator_Hardware_History.json` |
| Linux | `/home/用户名/.config/SeeSharp/JYTEK/AIGuideGenerator_Hardware_History.json` |

## 常见问题

### Q: 查找不到驱动？

- 确认硬件驱动已正确安装到驱动根目录
- 检查输入的型号数字是否正确
- 确认JYPEDIA文件版本是否匹配

### Q: 范例树为空？

- 确认范例已安装到范例根目录
- 范例目录名需同时包含型号数字和"Example"关键字

### Q: 历史记录丢失？

历史记录保存在用户数据目录，重装系统或更换用户可能导致丢失。
