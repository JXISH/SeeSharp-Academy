## 双螺旋桨声音数据

### 采集条件

办公室环境，用JY USB-1202两个通道，连接2个不同灵敏度的IEPE麦克风，采集2个船模螺旋桨运转声音。采样率25,600S/s，采样长度都是4秒钟。

用于展示数字信号处理的功能。

### 样本

前缀：20260224_Dual Propeller_

| 文件                    | 电机工况                           |
| ----------------------- | ---------------------------------- |
| Up-down 4 sec           | 双电机提速、降速过程               |
| High Speed 4 sec        | 双电机恒定高速                     |
| High Speed_one abnormal | 一个电机受阻尼降速，末尾带刮擦异响 |
| Low Speed 4 sec         | 双电机恒定低速1                    |
| 2nd Low Speed_Dual      | 双电机恒定低速2                    |
| 2nd Low Speed_Single    | 单点击恒定低速2                    |

### 提示

* 用16384点FFT分析可以清晰看到两个电机在相同工况下发出的声音频率不同。

* 长FFT对变速情况不适用。

* 建议试验联合时频分析观察变速下频率分量情况，用基于模型的频率分析解析短时样本的多频率分量。
