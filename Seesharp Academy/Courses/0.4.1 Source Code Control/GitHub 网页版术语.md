## GitHub 网页版・术语卡片（中英极简版）

### 🏠 仓库

- **Repository / Repo** — 仓库：项目总文件夹
- **Public** — 公开：所有人可见
- **Private** — 私有：仅自己可见

### 🌿 分支

- **Branch** — 分支：独立开发线
- **Main** — 主分支：稳定正式版
- **Create branch** — 新建分支
- **Delete branch** — 删除分支

### 📝 提交

- **Commit** — 提交：保存一次改动
- **Commit message** — 提交说明
- **History** — 历史：查看修改记录

### 🤝 协作核心

- **PR / Pull Request** — 合并请求
- **Merge** — 合并：把分支合入主分支
- **Conflict** — 冲突：代码改同一行
- **Review** — 代码审查
- **Approve** — 审核通过
- **Comment** — 评论

### 🔄 同步

- **Push** — 推送：上传到 GitHub
- **Pull** — 拉取：下载最新代码
- **Fetch** — 获取：只下载更新，不合并

### 🐛 任务

- **Issue** — 问题 / 任务
- **Label** — 标签
- **Close** — 关闭

### 🔐 权限

- **Star** — 收藏
- **Fork** — 复刻仓库
- **Collaborator** — 协作者
- **Settings** — 设置
- **Branch protection** — 分支保护

### 📁 文件

- **README.md** — 项目说明
- **.gitignore** — 忽略不上传的文件
- **Edit** — 编辑
- **Upload files** — 上传文件

### 🔀 合并方式

- **Merge commit** — 普通合并
- **Squash and merge** — 压缩合并（推荐）
- **Rebase and merge** — 变基合并
