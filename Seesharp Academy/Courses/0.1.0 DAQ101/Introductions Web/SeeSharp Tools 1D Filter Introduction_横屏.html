二、基本概念 Fundamental Concepts

2.1 FIR vs IIR 滤波器

FIR 滤波器 Finite Impulse Response

有限脉冲响应：输出仅依赖输入信号，无反馈。

  • 天然稳定（无极点）
  • 可实现严格线性相位
  • 阶数通常较高
  • 系数：仅 b[]（分子）
// FIR: y[n] = b[0]*x[n] + b[1]*x[n-1] + ... + b[N]*x[n-N] double[] coef_b; // FIR滤波器系数b，分子系数

IIR 滤波器 Infinite Impulse Response

无限脉冲响应：输出同时依赖输入和历史输出（有反馈）。

  • 需要稳定性验证（极点须在单位圆内）
  • 相位非线性
  • 阶数低，计算效率高
  • 系数：b[] + a[]（分子+分母）
// IIR: y[n] = Σb[k]*x[n-k] - Σa[k]*y[n-k] double[] coef_b; // 分子系数 double[] coef_a; // 分母系数

三、滤波器设计 Filter Design

FIR 滤波器设计方法

Kaiser 窗函数法 Kaiser Window

基于窗函数的经典方法，通过 β 参数控制主瓣宽度和旁瓣衰减的折中。

适用场景：对阻带衰减要求明确、过渡带不需要最窄的情况

等纹波法 Equiripple (Parks-McClellan)

最优化方法，通带和阻带误差均匀分布，给定阶数下过渡带最窄。

适用场景：对过渡带宽度有严格要求，需要最优滤波器

FIR 设计代码示例

// 创建滤波器实例 EasyClassicFilter filter = new EasyClassicFilter(); // 设计 FIR 低通滤波器 // 参数：设计方法, 滤波器类型, 通带波动(dB), 阻带抑制(dB), 采样率, // 通带频率, 截止频率 filter.DesignFIRFilter( FIRDesignMethod.Equiripple, // 设计方法: 等纹波法 FIRBandType.Lowpass, // 滤波器类型: 低通 0.01, // 通带波动: 0.01 dB 60.0, // 阻带抑制: 60 dB 10000, // 采样率: 10000 Hz 1000, // 通带频率: 1000 Hz 1500 // 截止频率: 1500 Hz );
AI编程提示词示例："设计一个FIR低通滤波器，采样率10kHz，通带截止1kHz，阻带起始1.5kHz，阻带衰减60dB，使用等纹波法。"

EasyClassicFilter filter = new EasyClassicFilter(); // 设计 IIR 带通滤波器（椭圆法） filter.DesignIIRFilter( IIRDesignMethod.Elliptic, // 设计方法: 椭圆滤波 IIRBandType.Bandpass, // 滤波器类型: 带通 0.5, // 通带波动: 0.5 dB 40.0, // 阻带抑制: 40 dB 44100, // 采样率: 44100 Hz 500, // 通带下边界: 500 Hz 200, // 阻带下边界: 200 Hz 4000, // 通带上边界: 4000 Hz 6000 // 阻带上边界: 6000 Hz );

SOS 级联二阶节设计 Second-Order Sections

已知阶数时可直接指定，使用 SOS 形式以提高数值稳定性：

// 已知阶数的 SOS 设计 filter.DesignSOSFilter( 4, // 滤波器阶数: 4阶 10000, // 采样率: 10000 Hz new double[] { 1000 }, // 通带频率数组 3.0, // 通带波动: 3 dB double.NaN, // 阻带抑制(Butterworth忽略) IIRBandType.Lowpass, // 类型: 低通 IIRDesignMethod.Butterworth // 方法: 巴特沃思 );

陷波/峰频滤波器 Notch & Peak Filter

陷波滤波器 Notch

在特定频率处深度衰减信号，常用于去除工频干扰（50/60Hz）。

峰频滤波器 Peak

仅让特定频率附近的信号通过/增强，用于频率选择。

四、滤波器分析 Filter Analysis

波特图 Bode Plot

显示滤波器的幅频响应和相频响应，是验证设计正确性的主要工具。

  • bodeMag：幅度响应 (dB)
  • bodePhase：相位响应 (度)
  • bodeFreq：频率轴 (Hz)

冲激响应 Impulse Response

滤波器对单位脉冲信号的时域输出，揭示滤波器的时域特性。

  • impulseResponseH：响应幅度
  • impulseResponseT：时间轴
// 设计完成后，分析结果自动计算 double[] magnitude = filter.bodeMag; // 幅度 (dB) double[] phase = filter.bodePhase; // 相位 (度) double[] freq = filter.bodeFreq; // 频率轴 (Hz) double[] impulseH = filter.impulseResponseH; // 冲激响应 double[] impulseT = filter.impulseResponseT; // 时间轴 // 分析长度可设置（默认1024点） filter.FrequencyResponseAnalysisLength = 2048;

// 基本滤波（自动判断FIR或IIR） double[] output = filter.Filtering(inputSignal, reset: true); // SOS级联滤波（更稳定，推荐用于高阶IIR） double[] output = filter.FilteringSOS(inputSignal, reset: true); // 连续数据块滤波（保持状态） double[] block1 = filter.Filtering(data_chunk1, reset: true); // 首块重置 double[] block2 = filter.Filtering(data_chunk2, reset: false); // 后续保持状态
实用建议：高阶IIR滤波器（>8阶）建议使用 FilteringSOS 方法，SOS（二阶节级联）形式可避免数值溢出问题。

5.3 关键概念：reset 参数

reset = true（重置）

清空历史状态，从零初始条件开始滤波。适用于：

  • 全新信号的首次处理
  • 非连续的独立数据段

reset = false（保持状态）

保留上次滤波的历史状态，实现无缝连续滤波。适用于：

  • 实时流式数据处理
  • 分块连续采集的信号
