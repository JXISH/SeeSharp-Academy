基础频谱分析 Fundamental Spectrum Analysis

本页对照 SeeSharpTools 类库 SeeSharpTools.JY.DSP.Fundamental.Spectrum 静态类， 以图文 + 动手实验的形式介绍频谱分析的核心概念，帮助你用中文向 AI 助手准确描述频谱分析需求。

This guide mirrors the public API of the SeeSharpTools Spectrum static class and shows how window function, sample length, and sample rate shape your spectrum.

实数输入 Real Input 复数输入 Complex Input 功率谱 PowerSpectrum dBFS 谱 窗函数 Window
面向 AI 编程的提示思路 / Prompting Tips:
1. 明确输入类型（实数 double[] / 复数 Complex[]）和采样率 Fs。
2. 指明所需谱线数 spectralLines（即输出数组长度）和窗类型 WindowType。
3. 说明输出单位（V / V² / dBV / dBm / dBFS）和是否需要 PSD 功率谱密度。
例: "用 SeeSharpTools 计算一个采样率 10 kHz、长度 1024 点的实数信号功率谱，使用汉宁窗，输出 dBV 单位。"

交互实验台 Interactive Lab

三个实验分别揭示窗函数、采样长度 N、采样率 Fs 对频谱的影响。所有图表实时计算。

实验 1: 窗函数对比 Window Type Comparison

采样率 Fs = 2048 Hz　长度 N = 1024　Δf = 2.000 Hz　ENBW = 1.50 bins
矩形窗主瓣最窄但旁瓣最高（-13 dB）；汉宁/海明等加窗后旁瓣下降但主瓣展宽； 平顶窗主瓣最平坦，适合幅度精确测量。
Rectangle: narrowest main lobe, worst leakage. Flat-top: best amplitude accuracy, widest main lobe.

实验 2: 采样长度 N Sample Length

Fs = 1024 Hz　N = 512　谱线数 = 256　Δf = 2.000 Hz
N 越大，频率分辨率 Δf = Fs/N 越细，但计算量越大、时间分辨率越差。 SeeSharpTools 中 spectralLines = N/2，FFT 实际点数 = 2 × spectralLines。

实验 3: 采样率 Fs 与混叠 Sampling Rate & Aliasing

最大可观测频率 Fmax = Fs/2 = 1000 Hz　实际显示 = 700 Hz
奈奎斯特定理 Nyquist: 当信号频率 f > Fs/2 时会发生混叠， 频率折叠显示为 |f − round(f/Fs)·Fs|。请试将信号频率调高至 Fs/2 以上观察折叠现象。

进阶: 输出点数与频率映射 Advanced: Output Length & Frequency Mapping

SeeSharpTools 在 AdvanceRealFFT 内部按 fftsize = 2 × spectralLines 计算实数 FFT， 实数输入仅返回单边谱。复数输入则返回完整双边谱。下表汇总四种 I/O 组合的输出点数与频率位置。

输入 Input 输出 Output N 偶 Even N 奇 Odd k=0 频率 末点 Last 中点 Middle
实数 Real 实数幅度谱
Real magnitude (single-sided)		 N/2 + 1 (N+1)/2 0 (DC) 偶: Fs/2 (Nyquist)
奇: ((N-1)/2)·Fs/N		 Fs/4
实数 Real 复数谱 (rfft, single-sided) N/2 + 1 (N+1)/2 0 (DC, 仅实部) 偶: Fs/2 (Nyquist, 仅实部)
奇: ((N-1)/2)·Fs/N		 Fs/4
复数 Complex 复数谱 (full, two-sided) N N 0 (DC) (N-1)/N · Fs ≈ Fs (即 −Δf) 偶: Fs/2 (折回)
奇: ((N-1)/2)·Fs/N
关键结论 / Key Takeaways:
① 实数输入有共轭对称性，只需返回 0~Fs/2，称为单边谱，长度 ⌊N/2⌋+1。
② 复数输入无对称性，需保留全部 N 个 bin（双边谱），bin k 对应频率 k·Fs/N， 其中后半部分 (k > N/2) 代表负频率 (k−N)·Fs/N。
③ N 偶数时存在精确的 Nyquist bin (Fs/2)，N 奇数时不存在。
④ SeeSharpTools 的 AdvanceRealFFT 在 0 频做了 ÷√2 校正以恢复 RMS 标度。

术语表 Glossary

FFT 快速傅里叶变换 / Fast Fourier Transform
DFT 离散傅里叶变换 / Discrete Fourier Transform

谱线 Spectral Lines 单边谱输出 bin 数 ≈ N/2
ENBW 等效噪声带宽 / Equivalent Noise Bandwidth
CG 相干增益 / Coherent Gain (窗均值)
PSD 功率谱密度 V²/Hz / Power Spectral Density
RMS 均方根 / Root Mean Square
dBFS 相对满量程的 dB / dB Full Scale
Spectral Leakage 频谱泄漏（非整周期采样导致）
Aliasing 频率混叠（Fs < 2f 时折叠）
Nyquist Fs/2 最大可观测频率
Single-sided / Two-sided 单边谱 / 双边谱
Window 窗函数（Hanning/Hamming/Flat-Top…）
