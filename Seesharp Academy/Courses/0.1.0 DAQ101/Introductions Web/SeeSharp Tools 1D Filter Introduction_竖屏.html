基本概念 Concepts

FIR

有限脉冲响应
  • 无反馈
  • 天然稳定
  • 线性相位
  • 阶数较高

IIR

无限脉冲响应
  • 有反馈
  • 需验证稳定性
  • 非线性相位
  • 阶数低效率高
滤波器设计 Filter Design
FIR 设计方法

Kaiser 窗函数法 Kaiser Window

通过β参数控制主瓣/旁瓣折中，适合阻带衰减要求明确的场景。

等纹波法 Equiripple / Parks-McClellan

最优化方法，给定阶数下过渡带最窄。

// FIR低通滤波器设计 filter.DesignFIRFilter( FIRDesignMethod.Equiripple, // 等纹波法 FIRBandType.Lowpass, // 低通 0.01, // 通带波动 dB 60.0, // 阻带抑制 dB 10000, // 采样率 Hz 1000, // 通带频率 Hz 1500 // 截止频率 Hz );

Notch / Peak 滤波器

陷波(Notch)：在特定频率深度衰减，如去除50Hz工频。

峰频(Peak)：仅让特定频率通过/增强。

// 陷波滤波器去除50Hz干扰 filter.DesignNotchPeakFilter( 1000, // 采样率 50, // 中心频率 50Hz 30, // Q因子 PeakNotchType.Notch );
Q值：Q = 中心频率 / 带宽。Q越大，滤波越窄越精确。
SOS 级联设计 (高阶稳定)

已知阶数时直接设计SOS（二阶节级联），数值更稳定。

// 4阶Butterworth低通SOS设计 filter.DesignSOSFilter( 4, // 阶数 10000, // 采样率 new double[]{ 1000 }, 3.0, // 通带波动 double.NaN, IIRBandType.Lowpass, IIRDesignMethod.Butterworth );
滤波器分析 Analysis

波特图 Bode Plot

幅度响应(dB) + 相位响应(°)，验证设计是否满足需求。

double[] mag = filter.bodeMag; // 幅度dB double[] phase = filter.bodePhase; // 相位° double[] freq = filter.bodeFreq; // 频率Hz

冲激响应 Impulse Response

滤波器对单位脉冲的时域输出，揭示时域特性。

double[] h = filter.impulseResponseH; // 响应 double[] t = filter.impulseResponseT; // 时间轴
分析长度：FrequencyResponseAnalysisLength 默认1024点，可按需调整。
滤波处理 Filtering

Filtering()

通用滤波
  • 自动判断FIR/IIR
  • FIR直接卷积
  • IIR差分方程

FilteringSOS()

SOS级联滤波
  • 仅IIR
  • 数值更稳定
  • 推荐高阶使用
// 基本滤波 double[] output = filter.Filtering(input, reset:true); // SOS滤波（推荐高阶IIR） double[] output = filter.FilteringSOS(input, reset:true); // 连续流式数据：首块reset，后续保持状态 var y1 = filter.Filtering(chunk1, reset:true); var y2 = filter.Filtering(chunk2, reset:false);

reset 参数说明

true：清空状态，处理独立信号段
false：保持状态，实现连续流式滤波
