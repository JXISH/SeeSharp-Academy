1创建任务 Create Task AOTask

模拟输出任务 (AO Task) 管理硬件输出 — 把"硬件 + 通道 + 参数 + 触发"封装在一起，将数字信号转换为模拟电压。

类比： 按槽位号 (Slot Number) 创建 aoTask 对象，就像在板卡上"登记座位"。它知道操作哪张卡，但还不知道从哪路输出。
成员 Member说明 Description
JY5500AOTask(slot)按槽位号创建 AO 任务
Create AO task by Slot Number
SlotNumber设备管理器中的槽位号
Slot index from Device Manager

2添加通道 AddChannel

      
告诉任务"从哪一路输出、量程多大"。AddChannel 将物理引脚 (AO0~AO3) 映射到任务中。


      

        📌 AO 量程： JY5510 支持 ±10V 和 ±5V。选择 ±5V 量程时 16-bit DAC 分辨率更高 (≈0.15 mV/step)。
      


      

        
参数说明取值

        
          
chnId通道号
Channel ID		0–3 (5510)

          
rangeLow/High量程范围
Range		±10V / ±5V

3配置参数 Configure

      
设定输出模式 (AOMode)、更新率 (UpdateRate)、输出点数。


      

        AOMode 四种模式：
        
    
          
  • Single 单点 — 直接写 DAC 寄存器
    • 
          
  • Finite 有限 — 输出 N 点后停止
    • 
          
  • ContinuousWrapping 循环 — 循环输出缓冲区
    • 
          
  • ContinuousNoWrapping 非循环 — 可实时追加
    • 
        

      


      

        
属性说明

        
          
Mode输出模式 AOMode

          
UpdateRate更新率 Sa/s (max 2.86M)

          
SamplesToUpdateFinite 模式输出点数

4写入数据 Write Data

      
将波形数据写入缓冲区。ContinuousWrapping 模式必须先 WriteData 后 Start。


      

        ⚠️ 顺序： Wrapping 模式先写后启动！否则 → BufferDataDownflow 异常。
      


      

        
方法说明

        
          
WriteData(double[], timeout)单通道写入波形

          
WriteData(double[,], timeout)多通道写入 (列存储)

          
WriteSinglePoint(val, ch)Single 模式直接输出

5启动与停止 Start & Stop

      
Start() 启动输出 → 硬件按更新率转换电压。结束后 Stop() + Channels.Clear()。


      

        触发 Trigger：
        
    
          
  • Immediate 立即 (默认)
    • 
          
  • Digital 数字 — PFI 边沿
    • 
          
  • Soft 软触发 — SendSoftwareTrigger()
    • 
        

      


      

        时钟同步 Clock：
        
    
          
  • Internal 内部 (默认) — 板卡自身振荡器
    • 
          
  • External 外部 — 接收 AI_SampleClock / PFI 实现同步
    • 
        

      


      

        
成员说明

        
          
Start()启动输出

          
Stop()停止输出

          
WaitUntilDone(timeout)Finite 模式等待完成

          
Channels.Clear()清除通道释放资源

          
Trigger.TypeImmediate/Digital/Soft

          
SampleClock.SourceInternal/External
